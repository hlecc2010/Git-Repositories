# Connectors

> Note: The use of the scripts requires the PowerShell Module Microsoft.PowerApps.Administration.PowerShell

+ [Add-PAFConnectorToBusinessDataGroup.ps1](./Add-PAFConnectorToBusinessDataGroup.ps1)

  Sets connector to the business data group of data loss policy

+ [Add-PAFCustomConnectorToPolicy.ps1](./Add-PAFCustomConnectorToPolicy.ps1)

  Adds a custom connector to the given group

+ [Get-PAFConnector.ps1](./Get-PAFConnector.ps1)

  Returns information about one or more custom connectors

+ [Get-PAFConnectorRoleAssignment.ps1](./Get-PAFConnectorRoleAssignment.ps1)

  Returns the connection role assignments for a user or a custom connection

+ [Remove-PAFConnector.ps1](./Remove-PAFConnector.ps1)

  Deletes the custom connector

+ [Remove-PAFConnectorFromBusinessDataGroup.ps1](./Remove-PAFConnectorFromBusinessDataGroup.ps1)

  Deletes a custom connector from the given DLP policy

+ [Remove-PAFConnectorRoleAssignment.ps1](./Remove-PAFConnectorRoleAssignment.ps1)

  Deletes a connector role assignment record

+ [Remove-PAFCustomConnectorFromPolicy.ps1](./Remove-PAFCustomConnectorFromPolicy.ps1)

  Deletes a custom connector from the given DLP policy

+ [Set-PAFConnectorRoleAssignment.ps1](./Set-PAFConnectorRoleAssignment.ps1)

  Sets permissions to the custom connectors
